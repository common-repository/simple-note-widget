You can use Simple Note Widget. It's a quick and simple note display on your Wordpress Website.Can use in a various way. To display instructions. Use it to display important notice on your website and so many other way.Take a look at DEMO - http://www.crayfishcreative.com/developer/simple-note/It's a quick development from us. We developed this one for a client and thought it might will be useful for other users of Wordpress Community.== Installation ==This section describes how to install the plugin and get it working.Download " simple-note-widget.zip" to your computer.Login WordPress as administratorClick on Plugins --> Add NewOn "Install Plugins" page, click on "Upload"Browse and select "simple-note-widget.zip" and click on "Install Now"You can activate it nowGoto ”Appearance->Widget”.Active the simple note widget Widget and Configure the setting.== Screenshots ==1.
